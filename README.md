# In Browser Chat

To load folder and bootstrap app please follow next steps:

```bash
$ git clone https://github.com/Rostyy/Chat.git to your local repository

$ npm install

$ bower install

$ npm start

should start on http://localhost:8000/

$ start with unit tests: npm run serve
```
